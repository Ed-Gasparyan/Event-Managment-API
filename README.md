Event Management API

Description:
This is a RESTful API built with .NET 8 and SQLite for managing events, venues, users, and tickets. It supports JWT authentication for secure login and registration, and allows admins to manage users, events, and venues.

Features
1)User registration and login with JWT authentication
2)Role-based access: Admin and Attendee
3)CRUD operations for Events and Venues
4)Ticket management: purchase, check availability, and revenue calculation
5)Paginated and filterable event listing
6)FluentValidation for input validation
7)Swagger UI for testing API endpoints
8)Error handling and logging all exceptions


Tech Stack
1).NET 8
2)C#
3)SQLite
4)Entity Framework Core
5)JWT Authentication
6)Swagger
7)FluentValidation
8)Error Handling and logging
